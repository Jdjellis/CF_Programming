Priority decisions

  • Front-squat strength deferred to class — Performance/WL squats heavy Mon/Wed/Sat. No competing barbell front squat; substituted low-CNS quad/knee support on Tue (the engine/pulling day).
  • Strict press kept as the protected independent lift (the box under-supplies pressing) — placed Thu AM: fresh, a gymnastics day (no press clash), and clear of Tue's Push Press.
  • Ring MU skill placed Mon/Wed/Sat for frequency (neurologically cheap). It is the first thing to flex if a day gets squeezed — strength is never cut for skill.
  • Triage if time/energy is short: PROTECT top sets → Performance class → quad/knee accessory → ring-MU skill.
