# Current focus blocks

The personal work that complements class. Run **one or two** concurrent focuses —
skill and strength don't interfere (different systems), so a skill block runs
alongside a strength emphasis. Each block names a drill/program file in
`references/drills/` and the week it's currently in; read this week's drills from
that file (see `policy.md` §4–§5).

Update the **current week** lines as the weeks pass (or infer the week from the
date / training log and confirm with me). When a block finishes, swap in the next.

---

## 1. Front Squat / Strict Press Strength — PROTECT, 2×/week

A 6-week strength emphasis on the two highest-leverage lifts. **Currently ~week 3
of 6.** This is priority #1; place on the freshest days, before conditioning.

- **Front squat** — program: `references/drills/squat-12wk-block.md`, **currently
  ~week 4 of 12**. Cue: *clean stand-up — 3s pause in the hole, stay vertical out
  of the front rack* (attacks the recovery-limited stand-up).
  - **Defer-to-class rule:** when the class already programmes heavy squats that
    week, do **not** add a competing barbell front squat. Defer the heavy stimulus
    to class and substitute the low-CNS **complement** below on a non-clashing class
    day (see `policy.md` §3).
  - **Complement (when class covers the squat):** quad / knee supporting work from
    `references/drills/knee-rehab.md` — ACCESSORY tier, low-CNS. Quad development +
    knee/tendon health; the class supplies the heavy squat stimulus.
- **Strict press** — the protected *independent* lift (the box under-supplies
  pressing, so this stays a real session on the freshest day, not deferred).
  Dedicated overhead-press volume: e.g. a top single @ RPE 8 then back-off volume
  at ~75%, plus a strict-HSPU progression and some half-kneeling DB press for
  overhead stability. Loads via `scripts/calc.py`.

## 2. Ring Muscle-Up Block — SKILL, 3×/week

A 6-week kipping ring-muscle-up skill block. **Currently ~week 3 of 6.** Program:
`references/drills/ring-mu-kipping.md`. Cue: **chase frequency, not load** —
neurologically cheap, so it survives low-readiness days (the "productive when
smashed" option) and is the first thing to flex out under time pressure. Read the
current week's drills from the program file.
